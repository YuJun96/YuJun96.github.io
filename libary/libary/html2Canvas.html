# html2canvas 实现网页截图+下载截图功能

  1. 下载相关第三方库
npm i html2canvas -S
1
  1. 在项目中引入
import html2canvas from 'html2canvas'
1
  1. 在项目中使用
 html2canvas(this.$refs.cardImg, { useCORS: true, backgroundColor: null, scrollY: 0 })
     .then(canvas => {
        var imgData = canvas.toDataURL('image/png')
        this.fileDownload(imgData)
     })
fileDownload (downloadUrl) {
   let aLink = document.createElement('a')
   aLink.style.display = 'none'
   aLink.href = downloadUrl
   aLink.download = '1.png'
      // 触发点击-然后移除
   document.body.appendChild(aLink)
   aLink.click()
   document.body.removeChild(aLink)
    },

  1. html2canvas 存在一些问题
/* 当要截取的dom元素中存在跨域的图片时,
   canvas不能进行toDataURL */
解决办法：需要后端设置图片允许跨域,并且前端设置 useCORS: true

/* 当有滚动条存在时，可能有空白问题 */
解决办法：设置scroll属性

/* 一些css3属性不能读取 */
例如： transform 垂直布局
