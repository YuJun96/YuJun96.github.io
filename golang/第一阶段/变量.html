# 输出格式

  1. printIn (这个一般用于输出单个变量的值,多个值也可以输出，只是不方便观看) printIn("num=",num) printIn("num=",num,",num1=",num1)

  2. printf(这里一般用于输出多个变量的值) printf("num=%d,num1=%d",num,num) 这里的 %d 表示的是整型的数据,注意这里 printf 不会换行，所以应该修改如下： printf("num=%d,num1=%d\n",num,num)

# 命名规范

  1. 变量名只能为数字,字母，下划线
  2. 数字不能作为变量名的开头
  3. 不能使用关键字作为变量名，例如 if else int
  4. 命名的规则一般分为三种: 小驼峰命名法: userName 大驼峰命名法: UserName 以下划线: user_name
