# 字符串类型

  1. 什么是字符串类型？由双引号引起来的字符
func main() {
  var name string = "aj"
  str:="a"
  //这里我们可以输出它的类型
  fmt.printf("%s\n",name)  值: aj
  fmt.printf("%T",name)  值: String
}
说明: %s,相当于一个占位符，代表的是字符串类型
      \n,表示换行
      %T，表示输出这个值得类型

# 类型转化

  1. 注意一般我们总是将低精度类型转化为高精度类型，这样不会造成精度丢失问题
  2. 几种类型转化的示例
  3. 为什么要进行类型转化？不同类型的数值不能进行运算 例如：int8 和 int16 两种类型的数就不能进行加法运算
var num float32 = 3.1212
a:=10
b:=3.92
c:=float64(a)*b
fmt.printf("%d\n",int(num))  //浮点转化为整型（不会四舍五入，只会保留整数部分）
fmt.printf("%f\n",float64(num))  //32转化为64

  const a int = 10
  // a = 20 这样程序会报错

# 格式化输出的总结

fmt.printf("%T",a)  // 输出数据类型
%t //输出的是bool型数据
%f //输出浮点型数据
%d //输出的是整型的数据
%c  // 不将字节类型转化为ascII
%s //输出字符串类型的数据
%p //输出该数据的地址（16进制）
